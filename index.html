About Me

From curiosity to engineering

I come from a non-technical background — no CS degree, no privileged head start. What I carried was an obsessive curiosity for how technology works, dating back to childhood.

Two years ago I opened a code editor for the first time. HTML → CSS → Python — each layer pulled me deeper. When I discovered that Python is the backbone of AI, the direction became clear. This was the field I wanted to master.

I enrolled in an AI & ML program with Intellipaat in collaboration with IIT Patna — one of India's premier institutions — receiving an alumni certificate and an IIT Patna student ID. I'm currently mastering end-to-end ML pipelines: data preprocessing, feature engineering, model training, and deployment.

What sets me apart is simple: I don't experience this as work. I overwork because I want to. That kind of intrinsic motivation compounds in ways that external pressure never can.
